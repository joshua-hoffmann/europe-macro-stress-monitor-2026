# Latest Observation Reading Guide

This guide explains how to read the latest official macro observation matrix in this repository.

The guide is descriptive only. It helps interpret data/latest_official_macro_stress_matrix.csv and does not add new data, retrieve new sources, or change the source boundary.

## What the matrix shows

The latest-observation matrix compares latest official Eurostat observations for Austria and EU27_2020 across six selected macro-stress channels.

The table is source-bounded. It should be read as a compact interpretation aid for the official observation table, not as a synchronized real-time dashboard.

## Frequency and timing limits

The matrix combines indicators with different reporting frequencies.

In the current matrix, 5 channels use annual observations and 1 channel uses a monthly observation. Latest observations should therefore be read as source-timestamped observations, not as synchronized real-time conditions.

## How to interpret the comparison

The matrix can support a bounded comparison of latest official observations between Austria and EU27_2020 for the selected indicators.

The comparison is useful for seeing which official indicators are available, which periods they refer to, and where frequency differences limit direct interpretation.

## What this guide does not provide

This guide does not provide forecasts, risk scores, country rankings, investment signals, policy recommendations, or crisis/recession predictions.

It is not a forecast model, crisis predictor, recession predictor, investment signal, policy recommendation tool, objective country ranking, validated early-warning model, synthetic risk score, traffic-light system, or production monitoring system.

## Source boundary

The guide depends only on the existing latest-observation matrix and official observation table in this repository.

It does not validate broader macroeconomic conditions, does not claim complete coverage, and does not replace source-specific interpretation of the underlying Eurostat datasets.
